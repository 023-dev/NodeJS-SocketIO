# Zoom

Zoom Clone using Node JS, WebRTC and Websockets.

Install

#make a package.json
npm init -y

#make README.md
touch README.md 

#install nodemon
npm i nodemon -D

#install babel
npm i @babel/core @babel/cli @babel/node -D

#make .gitignore file
touch .gitignore

#install babel/reset
npm i @babel/preset-env -D

#install express
npm i express

#install pug
npm i pug

#run project
npm run dev
