# Zoom

Zoom Clone using Node JS, WebRTC and Websockets.

Install

#make a package.json
<br>
npm init -y

#make README.md
<br>
touch README.md 

#install nodemon
<br>
npm i nodemon -D

#install babel
<br>
npm i @babel/core @babel/cli @babel/node -D

#make .gitignore file
<br>
touch .gitignore

#install babel/reset
<br>
npm i @babel/preset-env -D

#install express
<br>
npm i express

#install pug
<br>
npm i pug

#run project
<br>
npm run dev
